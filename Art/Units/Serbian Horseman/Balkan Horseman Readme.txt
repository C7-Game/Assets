Medieval Balkan Horseman
Created by Micaelus

There are three possible configurations of this unit. The unit can be armed with a sword or spear alone, or both the sword and spear. In order to use any of these configurations, delete the parenthetical portion of the filename of the .INI you wish to use (Sword), (Spear), or (Mixed).

The Sword configuration uses the following animations:
Default, Fidget, Fortify, AttackA, AttackB, AttackC, Victory, Death, and Run.

The Spear configuration uses the following animations:
Sp-Default, Sp-Fidget, Sp-Fortify, AttackD, AttackE, Sp-Victory, Sp-Death, and Sp-Run.

The Mixed configuration uses the following animations:
Sp-Default, Sp-Fidget, Sp-Fortify, AttackD, AttackT, AttackA, Sp-Victory, Death, and Sp-Run.

This unit is part of the Medieval Balkan Unit Pack, with the thread at http://forums.civfanatics.com/showthread.php?t=265320
